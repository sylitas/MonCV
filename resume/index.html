In 2017, I started my bachelor degree of Information Communication Technology (ICT) at University of Science and Technology of Hanoi (USTH). During second year of university, I did an internship at FPT Information System participated in a project named OFSAA. My main task was engaging with database involving mapping and testing.

Before graduating, I attended another project called Lung Cancer Care at ICTLab of university. In three months, I finished a website allowing doctors access into imaging data storage. It supports full logging interface and helps users with managing image data such as CT images or .dicom files, direct-viewing and simple image processing, for example, noting at each image, calculating distance from two point by milimeter, changing image color negative image, Etc.

Currently i'm working at HTC-ITC as a web-based software developer. Amis is the name of project that i am doing, which has triple A (AAA) computer sercurity, using Lightweight Directory Access Protocol (LDAP) for logging and also containing system for employee and customer management.

Because Javascript is a powerful programming language so i have experience in both frontend (HTML, CSS, JS ,PugJS ,VueJS) and backend (PHP, JS, NodeJs, ExpressJS) languages or framework. However, i am focusing on backend more than anything else.
